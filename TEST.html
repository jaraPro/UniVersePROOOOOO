Проверка ошибок

Откройте консоль браузера (F12 → Console) чтобы увидеть логи

Что проверить в консоли:

  • Ошибок при загрузке страниц не должно быть
  • CORS ошибок не должно быть
  • Undefined переменных не должно быть
  • universeAuth должен быть доступен глобально
  • localStorage должен содержать 'universe_users' и 'universe_currentUser'

Команды для консоли:

Введите в консоль для проверки: 

// Проверить класс
universeAuth

// Проверить текущего пользователя
universeAuth.currentUser

// Проверить всех пользователей
JSON.parse(localStorage.getItem('universe_users'))

// Проверить избранное
JSON.parse(localStorage.getItem('universe_favorites'))

// Проверить аутентификацию
universeAuth.isAuthenticated()

Известные особенности

⚠️ Важно для production:
  • Пароли хранятся в открытом виде (увеличить безопасность)
  • Нет HTTPS (добавить для production)
  • Нет backend (добавить Node.js + MongoDB)
  • localStorage только на одном браузере (добавить синхронизацию)

Несмотря на это, для разработки и демонстрации система работает отлично!

✨ Новые функции (v2.0)

🔒 Backend с JWT и bcrypt

Что добавлено: Полноценный Node.js сервер с MongoDB, JWT-аутентификация, bcrypt-хэширование паролей

Как запустить:

cd backend && npm install && npm start
Сервер запустится на http://localhost:5000

Тестирование: Все API эндпоинты доступны для тестирования

📧 Email/SMS верификация

Что добавлено: Верификация email при регистрации, восстановление пароля, SMS-уведомления

Настройка: Добавьте API ключи SendGrid и Twilio в .env файл

Тестирование: Регистрация отправляет email для подтверждения

🌐 Google OAuth 2.0

Что добавлено: Вход через Google-аккаунт в модальном окне

Настройка: Добавьте Google Client ID в backend/.env

Тестирование: Кнопка "Войти через Google" в форме входа

🔐 Двухфакторная аутентификация (2FA)

Что добавлено: 2FA с Google Authenticator, QR-коды, резервные коды

Тестирование: В профиле → Настройки → Настроить 2FA

Особенности: 10 резервных кодов для восстановления доступа

🌙 Темная тема

Что добавлено: Переключатель светлая/темная тема в профиле

Тестирование: В профиле нажмите кнопку луны/солнца в правом верхнем углу

Особенности: Тема сохраняется в localStorage

✨ Анимации и микро-интеракции

Что добавлено: Плавные переходы, hover-эффекты, анимации карточек

Тестирование: Наведите курсор на карточки университетов

Особенности: fadeIn, slideIn, pulse анимации
