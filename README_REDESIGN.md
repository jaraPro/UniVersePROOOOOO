# 🎓 UniVerse Pro - Система Поиска Университетов

## 📋 Обзор проекта

**UniVerse Pro** - это модернизированная платформа для поиска и сравнения университетов со всего мира. Полностью переделанная система с профессиональным интерфейсом, многоаккаунтностью и советанной регистрацией.

## 🚀 Как использовать

### **Для пользователя:**

1. **Главная страница** (`index2.html`):
   - Нажмите "Начать поиск" для перехода в каталог
   - Или нажмите "Вход/Регистрация" в правом углу

2. **Поиск университетов** (`index5.html`):
   - Введите название, выберите страну/город/рейтинг
   - Нажмите "Поиск" для отбора результатов
   - Добавляйте в избранное (❤️ кнопка)
   - Нажимайте "Подробнее" для информации об университете

3. **Профиль** (`profile.html`):
   - Нажмите на вашу аватарку в навбаре
   - **Вкладка "Профиль"** - просмотр информации
   - **Вкладка "Редактирование"** - изменение данных
   - **Вкладка "Избранное"** - управление любимыми университетами
   - **Вкладка "Настройки"** - опции аккаунта
   - **Переключение аккаунтов** - нажмите на другой аккаунт в боковой панели
   - **Добавить аккаунт** - кнопка "+ Добавить аккаунт" в боковой панели

### **Для разработчика:**

#### **Добавить регистрацию на любую страницу:**
```html
<!-- В head секцию -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<!-- Перед </body> -->
<script src="auth.js"></script>

<!-- Кнопка входа -->
<button onclick="universeAuth.openModal()">Войти/Регистрация</button>

<!-- Проверка аутентификации -->
<script>
  if (universeAuth.isAuthenticated()) {
    console.log('Пользователь:', universeAuth.currentUser);
  }
</script>
```

#### **API auth.js:**
```javascript
// Проверить текущего пользователя
universeAuth.currentUser // объект пользователя или null

// Проверить, авторизован ли
universeAuth.isAuthenticated() // true/false

// Открыть модальное окно
universeAuth.openModal()

// Закрыть модальное окно
universeAuth.closeModal()

// Регистрация (используется внутри)
universeAuth.register(name, email, phone, password)

// Вход (используется внутри)
universeAuth.login(email, password)

// Выход
universeAuth.logout()
```

// Копия одного пользователя из массива выше

## 🔒 Безопасность

⚠️ **Важно для production:**
1. Пароли хранятся в **открытом виде** в localStorage (использовано для простоты в разработке)
2. Для production нужно:
   - Использовать HTTPS
   - Хешировать пароли на backend (bcrypt)
   - Использовать JWT токены
   - Реализовать CORS
   - Добавить rate limiting

## 🚦 Как запустить

1. **Откройте** `index2.html` в браузере (главная страница)
2. **Нажмите** "Начать поиск" → попадете на `index5.html`
3. **Зарегистрируйтесь** через модальное окно
4. **Ищите** университеты с фильтрацией и сортировкой
5. **Добавляйте** в избранное (❤️)
6. **Переходите** в профиль для управления аккаунтами

---

## 💡 Примечания

- Все данные сохраняются в `localStorage` (браузер)
- При смене браузера или очистке кэша данные удалятся
- Для синхронизации между устройствами нужен backend
- Модальное окно можно открыть с любой страницы через `universeAuth.openModal()`
