   └─ Красивая страница с тремя вкладками
   └─ Вкладка 1: Просмотр информации об аккаунте
   └─ Вкладка 2: Редактирование профиля (имя, телефон, локация, биография)

   └─ Добавление университетов в избранное ⭐

   └─ Удаление из избранного
   └─ Быстрые ссылки на университеты
   └─ Сохранение в MongoDB

3. ✅ API ЭНДПОИНТЫ
   └─ POST /api/auth/favorites/add - добавление
   └─ DELETE /api/auth/favorites/:id - удаление
   └─ GET /api/auth/favorites - получение списка
   └─ PUT /api/auth/profile/update - обновление профиля

4. ✅ ГОТОВАЯ БИБЛИОТЕКА (favorites.js)
   └─ addToFavorites() - добавить в избранное
   └─ removeFromFavorites() - удалить
   └─ getFavorites() - получить список
   └─ isFavorited() - проверить наличие

5. ✅ ПРИМЕР СТРАНИЦЫ ВУЗа (university-example.html)
   └─ Полная информация о университете
   └─ Кнопка добавления в избранное
   └─ Красивый интерфейс
   └─ Рефевые отзывы и статистика
   └─ Контактная информация

6. ✅ ОБНОВЛЕНЫ РЕДИРЕКТЫ
   └─ После регистрации → profile.html ✨
   └─ После входа → profile.html ✨
   └─ После Google OAuth → profile.html ✨

🚀 КАК ЗАПУСТИТЬ:

# 1. Установите зависимости
npm install

# 2. Убедитесь, что MongoDB запущена
mongod

# 3. Запустите сервер
npm run dev

# 4. Откройте в браузере:
Регистрация: http://localhost:5000/register.html
Вход: http://localhost:5000/login.html
Пример ВУЗа: http://localhost:5000/university-example.html

🧪 ТЕсТИРОВАНИЕ:

Тест 1: Регистрация
─────────────────
1. Откройте register.html
2. Заполните форму (пароль минимум 8 символов)
3. Нажмите "Зарегистрироваться"
4. Должны перейти на profile.html ✅

Тест 2: Редактирование профиля
────────────────────────────────
1. На profile.html откройте "Редактирование"
2. Измените имя, телефон, локацию
3. Нажмите "Сохранить изменения"
4. Обновите страницу
5. Данные должны остаться ✅

Тест 3: Добавление в избранное
────────────────────────────────
1. Откройте university-example.html
2. Нажмите "Добавить в избранное"
3. Кнопка изменится на "⭐ В избранном"
4. На profile.html в разделе "Избранное" появится МГУ ✅

Тест 4: Удаление из избранного
────────────────────────────────
1. На profile.html откройте "Избранное"
2. Нажмите "Удалить"
3. Университет исчезнет ✅

🎨 ИНТЕГРАЦИЯ НА ДРУГИЕ СТРАНИЦЫ:

Чтобы добавить функциональность избранного на свои страницы:

1. Добавьте скрипт:
   <script src="favorites.js"></script>

2. Добавьте кнопку:
   <button onclick="toggleFavorite('uni-id', 'Название', 'https://...', 'Город')">
     Добавить в избранное
   </button>

3. Инициализируйте при загрузке:
   window.addEventListener('load', async () => {
     const isFav = await isFavorited('uni-id');
     if (isFav) {
       document.getElementById('favBtn').classList.add('favorited');
     }
   });
